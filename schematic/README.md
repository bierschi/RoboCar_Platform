## GPIO (BCM) / Bus Mapping

**Gearmotor** <br>
GPIO 12: PWM Pin <br>
GPIO 5: Direction Pin

**HC-SR04** <br>
GPIO 23: Trigger Pin<br>
GPIO 24: Echo Pin<br>

**MPU9250** <br>
I2C Bus address: 0x68

**SteeringServoMotor** <br>
I2C Bus address: 0x40

**CameraServoMotor** <br>
I2C Bus address: 0x70
